# PyNmap_Scanner

A professional GUI-based network scanning tool built with Python, Tkinter, and Nmap.

PyNmap_Scanner allows users to scan targets, detect open ports, identify running services, save scan reports, and maintain scan history through an interactive graphical interface.

---

## Features

- GUI-based network scanner
- Target IP / Domain scanning
- Custom port range selection
- Multiple scan modes
  - Quick Scan
  - Service Detection
  - Aggressive Scan
- Dark themed professional interface
- Scan history management
- Export scan reports
- Real-time scan output display

## Installation

### Clone Repository

```bash
git clone https://github.com/amitraj231226-cmyk/PyNmap_Scanner.git
cd PyNmap_Scanner
```

### Install Dependencies

```bash
pip install python-nmap
```

### Install Nmap

Download and install Nmap from:

https://nmap.org/download.html

---

## Run the Application

### CLI Version

```bash
python scanner.py
```

### GUI Version

```bash
python scanner_gui.py
```

## How It Works

1. Enter target IP address or domain
2. Enter custom port range
3. Select scan type
4. Start scan
5. View real-time results
6. Save scan reports
7. Access scan history
